# Contributing

Thanks for helping improve Gmail Footer Cleaner!

1. Fork → feature branch → commit → PR
2. Keep it zero-permission and lightweight
3. Test via `chrome://extensions` (enable in Incognito to verify isolation)
4. Follow Conventional Commits (e.g., `feat:`, `fix:`, `docs:`, `chore:`)

## Releasing
- Update `manifest.json` version
- Tag (e.g., `v1.0.2`) and push tags
- GitHub Action will attach a ZIP to the release
